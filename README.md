## Getting started

### Requirements

- Node >= 20.x, see **.nvmrc**
- pnpm > 9.x

### Installation

#### Install packages and start

`pnpm install && pnpm dev`

### Build and Deploy

#### Build

`pnpm build`

#### Start the built server

`pnpm build && pnpm start`
